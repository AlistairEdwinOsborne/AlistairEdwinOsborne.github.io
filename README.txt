Visualisation of Machine Learning
M.E.N.A.C.E system

This system does somewhat lose functionality when run locally, as such these files are provided as a reference.

Best performance is acheived when running on Windows OS, using Chrome or Firefox 64-bit Browsers any other platform compatability issues have been explained with the report.
It is suggested to keep a developer console open alongside using the system, as to get a better grasp of its inner workings.
